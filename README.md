aEditor
=======

##汇编实现的简易图形编辑器，可满足课程设计的要求

###实现的功能
1. 下拉菜单点击事件；
2. 文件新建、打开，保存；
3. 绘制直线、多边形、圆；
4. 改变画笔颜色；
5. 矩形框选择和删除。

###说明
- *文件打开默认加载当前目录下NAME1.BMP图片，而文件保存至该目录下命名为NAME2.BMP。（只支持BMP图片格式）*
